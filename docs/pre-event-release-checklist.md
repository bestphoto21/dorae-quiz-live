# 행사 전 최종 릴리즈 체크리스트

실제 행사 투입 전 마지막으로 확인하는 릴리즈 체크리스트입니다. 이 문서는
기능 추가가 아니라 배포, 환경, 운영 준비 상태를 확인하기 위한 문서입니다.

## 1. 코드/배포

- [ ] `git status`가 clean입니다.
- [ ] `npm.cmd run lint`가 통과했습니다.
- [ ] `npm.cmd run build`가 통과했습니다.
- [ ] Vercel 최신 배포가 Ready 상태입니다.
- [ ] 운영에 사용할 최신 커밋을 확인했습니다.
- [ ] 배포 URL이 운영자에게 공유되었습니다.

## 2. Supabase

- [ ] 필요한 migration이 모두 적용되어 있습니다.
- [ ] `events` 테이블에 참가자 화면 설정 컬럼이 있습니다.
- [ ] `events` 테이블에 스크린 화면 설정 컬럼이 있습니다.
- [ ] 테스트 `event_code`와 실제 행사 `event_code`를 구분했습니다.
- [ ] 이번 릴리즈에서 RLS 변경이 없음을 확인했습니다.
- [ ] 이번 릴리즈에서 auth/session 구조 변경이 없음을 확인했습니다.

## 3. 관리자

- [ ] 관리자 로그인이 가능합니다.
- [ ] 통합 운영 콘솔에 접속할 수 있습니다.
- [ ] 운영 체크리스트를 확인했습니다.
- [ ] 행사 설정을 확인했습니다.
- [ ] 참가자 화면 제목, 설명, 기능 ON/OFF를 확인했습니다.
- [ ] 스크린 화면 제목, 안내 문구, 로고 표시 여부를 확인했습니다.
- [ ] 운영 로그 페이지 접근을 확인했습니다.

## 4. 참가자

- [ ] QR 입장이 가능합니다.
- [ ] 모바일 화면이 정상 표시됩니다.
- [ ] 참가자 세션이 새로고침 후 유지됩니다.
- [ ] 설문 제출이 가능합니다.
- [ ] 제출 완료 안내가 정상입니다.
- [ ] 제출 후 수정 불가 안내가 표시됩니다.
- [ ] Q&A ON/OFF 동작이 정상입니다.
- [ ] 퀴즈/Q&A/설문/럭키드로우 안내 표시가 행사 설정과 일치합니다.

## 5. 스크린

- [ ] 대기 화면이 정상입니다.
- [ ] QR 화면이 정상입니다.
- [ ] 설문 화면이 정상입니다.
- [ ] Q&A 화면이 정상입니다.
- [ ] 럭키드로우 준비 화면이 정상입니다.
- [ ] 럭키드로우 당첨 화면이 정상입니다.
- [ ] 효과음을 사용할 경우 스크린 PC에서 오디오를 확인했습니다.
- [ ] 로고 표시 여부가 설정과 일치합니다.
- [ ] 화면 전환이 크게 깜빡이지 않습니다.

## 6. 데이터

- [ ] 테스트 참가자 데이터 초기화 여부를 확인했습니다.
- [ ] 테스트 설문 응답 초기화 여부를 확인했습니다.
- [ ] 테스트 Q&A 질문 초기화 여부를 확인했습니다.
- [ ] 테스트 당첨자 기록 초기화 여부를 확인했습니다.
- [ ] 실제 행사 전 데이터 상태를 확인했습니다.
- [ ] CSV 다운로드가 가능합니다.
- [ ] CSV에 `participant_id`, phone, email, raw `screen_payload`가 없습니다.

## 7. 현장 장비

- [ ] 스크린 해상도를 확인했습니다.
- [ ] 브라우저 확대 비율을 확인했습니다.
- [ ] 오디오 출력 장치를 확인했습니다.
- [ ] 행사장 네트워크를 확인했습니다.
- [ ] 예비 인터넷 연결 방법을 준비했습니다.
- [ ] 예비 운영자 PC 또는 예비 로그인 수단을 준비했습니다.
- [ ] 스크린 PC 절전, 알림, 자동 업데이트 설정을 확인했습니다.
